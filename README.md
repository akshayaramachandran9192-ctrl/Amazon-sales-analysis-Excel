# Amazon Sales Analysis Report 
Developed an interactive Amazon Sales Analysis Report using Advanced Excel to transform messy e-commerce data into actionable business strategies. Cleaned and structured the raw dataset using automated text-functions and transformations, then built dynamic Pivot Dashboards to monitor key performance indicators (KPIs). The final report delivers high-impact insights into top-performing categories, monthly revenue trends, regional demand distribution, and order fulfillment status to optimize inventory management and drive sales growth."

## 📌 Project Overview
This project focuses on analyzing Amazon sales data to uncover key insights regarding product performance, sales trends, regional demand, and customer purchasing behavior. Using **Advanced Excel**, the raw dataset was cleaned, structured, and analyzed to help businesses make data-driven decisions to optimize inventory and maximize revenue.

## 🛠️ Tech Stack & Tools Used
* **Tool:** Microsoft Excel
* **Features Used:** Power Query (if used for cleaning), XLOOKUP/VLOOKUP, Pivot Tables, Pivot Charts, Conditional Formatting, Data Validation, and Interactive Dashboard.

## 🧽 Data Cleaning & Preparation
Before diving into analysis, the raw Amazon dataset underwent a rigorous data cleaning process to ensure accuracy:
* **Handling Missing Values:** Filled or removed null values in critical columns like Customer ID, Ratings, or Sales.
* **Data Standardization:** Corrected date formats (YYYY-MM-DD), removed duplicate rows, and trimmed unwanted spaces.
* **Data Transformation:** Created new calculated columns such as `Total Revenue`, `Profit Margin`, `Order Month`, and `Delivery Status` using Excel formulas.

## 📊 Key Insights Shared (Objectives)
Through Pivot Tables and Charts, this report answers the following business questions:
1. **Top Performing Categories:** Which product categories and sub-categories generate the highest sales and profit?
2. **Sales Trends:** How do sales fluctuate over months or quarters? (Identifying peak seasons like Diwali/Christmas sales).
3. **Regional Analysis:** Which states/cities are driving the maximum orders?
4. **Order Status:** What percentage of orders are successfully delivered vs. cancelled or returned?
